# Career Booster

Your personal AI career manager — from CV to hired.

Career Booster covers the complete job search pipeline: profile building, CV parsing and optimization, market analysis, job discovery, application packaging, LinkedIn optimization, and employer outreach.

## Skills

| Skill | Slash command | What it does |
|---|---|---|
| profile-setup | /setup | Build your career profile from scratch |
| cv-parser | *(upload a CV)* | Extract and store all information from an existing CV |
| cv-optimizer | /optimize-cv | Improve your CV for ATS and recruiter attractiveness |
| cv-builder | /build-cv | Generate a complete CV from your stored profile |
| cv-tailor | /tailor-cv | Customize your CV for a specific job posting |
| market-analyzer | /analyze-market | Research job demand, salaries, and skills for your target role |
| job-finder | /find-jobs | Search job boards and return ranked listings with suitability scores |
| job-analyzer | *(paste a job)* | Score a specific posting against your profile |
| application-packager | /apply | Generate a tailored CV + cover letter + email for one job |
| email-writer | /write-email | Write a personalized outreach email to an employer or recruiter |
| linkedin-auditor | /audit-linkedin | Score your LinkedIn profile section by section |
| linkedin-optimizer | /optimize-linkedin | Rewrite LinkedIn sections for recruiter visibility |
| linkedin-outreach | /find-connections | Find and message relevant LinkedIn contacts |
| application-tracker | /tracker | View and update your full application history |

## Pipeline

Career Booster tracks where you are in the job search process:

```
Setup → CV intake → CV optimization → Market analysis → Job discovery
→ Job analysis → Application packaging → Apply → LinkedIn → Outreach → Tracking
```

After each step, Career Booster suggests the next logical action.

## Connectors

For full functionality, connect the following:

- **Gmail** — send outreach emails to employers and recruiters
- **Browser** — search job boards, access LinkedIn, research companies

## First run

On first use, Career Booster automatically runs /setup to build your profile. On subsequent sessions, it loads your profile silently and gives a one-line status update.

## Multilingual support

Career Booster detects the language of every job posting and matches all application documents to it. Separate CV and cover letter versions are maintained per language and per target market. Local CV conventions (RODO for Poland, one-page for Israel, photo norms for Germany, etc.) are applied automatically.
